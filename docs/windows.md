# Run the code in Windows (via WSL)

Since the `cs50.h` library is designed for Unix-based systems (Linux/macOS), the best and most stable way to run this calculator on Windows is by using **WSL (Windows Subsystem for Linux)**.

## Steps

1. Open your Windows Terminal or PowerShell as Administrator and install WSL (Ubuntu) by running:
   ```bash
   wsl --install
   ```

2. Restart the computer if promted
3. Open your newly installed Ubuntu terminal from the Windows Start Menu
4. Now, simply follow the exact same steps in our [Ubuntu Guide](ubuntu.md).
